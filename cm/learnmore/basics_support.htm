What browser versions does the platform support?

The platform supports the following web browsers:

  • Mozilla Firefox 5.0+
  • Internet Explorer 8.0+
  • Google Chrome 13+
  • Safari 4+

Note: If you are currently using an unsupported browser, it is recommended that you upgrade your browser to one of the supported versions to ensure optimum performance of the platform.

Who do I contact if I have technical questions?

The following approaches can be used to resolve technical questions:

  • Most technical questions can be addressed by posting discussions or tickets on the Boards of the respective API or app.
  • You can query the site using the Support / Knowledge base search function on the Learn More page.
  • You can also submit an email to the email address listed on the Support page which is accessible by clicking ? in the top navigation. Please provide your Name, Email, Subject, and Question. You'll receive an acknowledgement email and one of our support staff will contact you.
