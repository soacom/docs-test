GET /api/users/{UserID}/logindomain Method

Returns information about the login domain for the specified user.

Authorization Roles/Permissions: Authorized User or Site Admin.

Request Parameters

Parameter Parm Type Data Type Required Description
UserID Path string Required The unique ID for a specific user.
cache Query boolean Optional Indicates whether the information should be cached. If you don't want the domain information to be read from cache, specify false. If not specified, defaults to true. For more information about caching on the platform, see How the Community Platform API Uses Caching.

Back to top

Response

If successful, this method returns information about the login domain for the user specified in the request message.
