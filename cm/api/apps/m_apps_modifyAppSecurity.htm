PUT apps/versions/{AppVersionID}/securitymechanisms Method

Changes the security mechanism selected for the specified app.

There are two app security mechanisms available:

  • SharedSecret
  • KeyInfo

For more information about security mechanisms on the platform, see API Authentication and Security on the Community Platform.

Authorization Roles/Permissions: App team member, Business Administrator
