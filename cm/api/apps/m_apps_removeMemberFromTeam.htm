DELETE apps/{AppID}/members/{UserID} Method

Removes a member from the team for the specified resource.

Authorization Roles/Permissions: App team member, Business Administrator, or Site Admin.

HTTP Method

DELETE

Back to top

URL

https://<hostname>/api/apps/{AppID}/members/{UserID}[?Comment={Comment}]

Back to top

Sample Request

In the example shown below, user user10021.open is removed from the app team for app10017.open with the comment, "Removing inactive members."

Sample request URL

https://<hostname>/api/apps/app16964.open/members/user13967.open?Comment=Removing%20Jonathan%20from%20the%20team%20at%20his%20request.

Sample request headers

DELETE /api/apps/app16964.open/members/user13967.open?Comment=Removing%20Jonathan%20from%20the%20team%20at%20his%20request. HTTP/1.1
Host: <hostname>
Accept: */*
Content-Type: application/json

Sample request body

Not applicable.

Request Parameters

Parameter Parm Type Data Type Required Description
AppID Path string Required The unique ID for the specified app.
UserID Path string Required The unique ID for a specific user. In this case it is the user being removed from the team.
Comment Query string Optional A comment entered by the user removing the team member.

Back to top

Response

If successful, this method returns HTTP status code 200, with the UserID of the removed team member.

Back to top

Sample Response

In the example below, the user listed in the response body has been removed from the team.

Sample response body
