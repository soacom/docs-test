POST /api/apps/versions/{RuntimeID}/keyinfo Method

Adds an app's public key or certificate (for runtime security).

Authorization Roles/Permissions: App team member, Business Administrator

HTTP Method

POST

URL

Sample Request

The example below shows a call to this operation.

Sample request URL

https://<hostname>/api/apps/versions/4iXro2Uxs2h4LviCshR9pK9w.acmecorp/keyinfo?wrapInHTML=true

Sample request headers

POST /api/apps/versions/4iXro2Uxs2h4LviCshR9pK9w.acmecorp/keyinfo?wrapInHTML=true HTTP/1.1
Host: <hostname>
Accept: text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8

Sample request body

The request body is the CSR certificate which is uploaded.

Request Parameters

InMultiPart Body multipart Required The CSR information.

Response

If successful, this method returns HTTP status code 200, with the runtime ID as confirmation that the operation completed successfully. The response is wrapped in HTML.
