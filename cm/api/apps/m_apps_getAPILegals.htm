GET /api/apps/versions/{RuntimeID}/apilegals Method

Returns the legal documents associated with an API that an app is connecting to.

Authorization Roles/Permissions: None.

Request Parameters

Parameter Parm Type Data Type Required Description
RuntimeID Path string Required Unique ID used at runtime by the app version.
APIVersionID Path string Required The unique ID for the specified API version.
State Query string Optional If a state is specified, only legal documents in that state are returned. Possible values: com.soa.status.active, com.soa.status.inactive. For more information, see Legal Document State.
Federation Query boolean Optional Indicates whether the current implementation is a federation. If not specified, defaults to true.

Back to top

Response

If successful, this method returns HTTP status code 200, with information about the legal agreement documents for the specified API version. The response is returned as an RSS feed in JSON format.
