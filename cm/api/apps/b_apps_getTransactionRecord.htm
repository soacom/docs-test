GET /api/apps/versions/{AppVersionID}/txlogs/{TransactionID} Method

Returns information about the specified transaction for an app version.

For additional information regarding the set of operations dealing with metric information, see Metrics on the Platform.

Authorization Roles/Permissions: App team member, Business Administrator, or Site Admin.

Request Parameters

Parameter Parm Type Data Type Required Description
AppVersionID Path string Required The unique ID for the specified app version.
TransactionID Path string Required The unique ID for a specific transaction.
TimeZone Query string Optional The time zone for the information. If a time zone isn't specified, the information is returned in GMT. For more information, refer to Working with Time Zones and Time Intervals.

Response

If successful, this method returns HTTP status code 200, with the transaction information in the requested format.
