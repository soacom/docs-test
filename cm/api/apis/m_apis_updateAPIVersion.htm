PUT /api/apis/versions/{APIVersionID}

Updates information about an API version.

Authorization Roles/Permissions: To complete this operation successfully, a user must be logged in and must be an Administrator for the API.

HTTP Method

PUT

Back to top

URL

https://<hostname>/api/apis/versions/{APIVersionID}

The example below adds a third tag to the API version.

Request Parameters

Parameter Parm Type Data Type Required Description
APIVersionID Path String Required The unique ID for the specified API version.
APIVersion Body APIVersion Required Includes information about an API version. The entire version, including the changed data.

Back to top

Response

If successful, this method returns HTTP status code 200, with the updated API version object.
