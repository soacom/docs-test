UserAgreement Object

Includes information about a user's agreement to a legal document associated with an API, including the ID of the document and the ID of the user who is making the agreement.

This object is used by the following operations:

  • POST /api/legals/agreements

The properties included in the UserAgreement object are listed below. Properties are optional unless otherwise specified.

Property Type Description
AgreementID long A unique ID for the legal agreement.
UserID string The unique ID for a specific user. Required in the request message when a user is accepting a legal agreement.
DocumentID string The unique ID for a specific legal document, such as an API licensing agreement. Required in the request message when a user is accepting a legal agreement.
AgreementScopeID string A unique ID for the scope of the legal agreement.
AgreementDate DateTime The date/time that the agreement was activated with the user.
