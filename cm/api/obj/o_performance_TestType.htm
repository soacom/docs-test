TestType Object

Contains information about a type of metric test.

This object is used by the following operations:

  • Operations that reference the Graph object

The properties included in the TestType object are listed below.

Property Type Description
Operator string The operator.
MetricType string The type of metric.
Value integer The value for the metric.
ValueUnit string The unit of measure for the value.
