ModifyAppSecurityRequest Object

Includes information about a request to modify the security information for an app.

This object is used by the following operations:

  • PUT apps/versions/{AppVersionID}/securitymechanisms

The properties included in the ModifyAppSecurityRequest object are listed below. All properties are required.

Property Type Description
SecurityMechanismSelected string The security mechanism selected for the app.
ActionName string The name of the workflow action; for example, group.membership.action.accept. For valid values, see Workflow Actions.
