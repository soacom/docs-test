SharedSecret Object

Includes information about an app's Shared Secret value.

This object is used by the following operations:

  • GET /api/apps/versions/{AppVersionID}/secret

The properties included in the SharedSecret object are listed below.

Property Type Description
SharedSecret string The encrypted shared secret value for the app.
