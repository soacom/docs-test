Special Workflow States and Workflow Tasks List

Description of special workflow state and workflow task list.

Workflow Management Workflow Library Workflow Developers Guide

Introduction

The Workflow states can be classified into two categories: A state that requires immediate attention, and a state that does not.

To mark a Workflow state as requiring immediate attention, you can prepend or append one or more "!" symbols to the name of the Workflow state (e.g., "Pending Approve!!!" or "!Pending Approve").

When a Workflow enters a state with this special markup, it will be added into the Workflow task list so user can be immediately notified via Workbenchâ€™s Workflow task portlet.
